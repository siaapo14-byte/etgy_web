# 益路同行 - 前端应用

## 技术栈

- Vue 3
- TypeScript
- Element Plus
- Pinia
- Vue Router
- Vite
- Axios（待后端接口完成后集成）

## 项目结构

```
frontend/
├── src/
│   ├── views/              # 页面组件
│   │   ├── Login.vue       # 登录页
│   │   ├── volunteer/      # 志愿者端页面
│   │   ├── college-admin/  # 学院管理员端页面
│   │   └── platform-admin/ # 平台管理员端页面
│   ├── layouts/            # 布局组件
│   ├── router/             # 路由配置
│   ├── stores/             # 状态管理
│   ├── utils/              # 工具函数
│   │   ├── mockData.ts     # 硬编码数据
│   │   └── api.ts          # API调用（待替换）
│   └── main.ts             # 入口文件
├── public/                 # 静态资源
└── package.json
```

### 角色说明

1. **志愿者端** (`/volunteer`)
   - 我的视频：查看、提交审核、上架/下架
   - 上传视频：创建视频草稿
   - 直播申请：申请和管理直播
   - 数据统计：查看个人数据

2. **学院管理员端** (`/college-admin`)
   - 数据概览：学院数据统计
   - 视频审核：审核志愿者提交的视频
   - 直播审核：审核直播申请
   - 志愿者管理：管理志愿者账号
   - 儿童账号管理：维护儿童账号
   - 审计日志：查看操作记录

3. **平台管理员端** (`/platform-admin`)
   - 全局仪表盘：全平台数据对比
   - 全局视频管理：兜底下架
   - 全局直播管理：封停/下架
   - 学院管理：创建和管理学院
   - 学院管理员管理：管理管理员账号
   - 儿童账号建档：批量导入儿童账号
   - 全局审计日志：查看全平台日志
   - 风险告警：处理高风险事件

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

## 构建生产版本

```bash
npm run build
```
