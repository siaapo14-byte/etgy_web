# 快速开始

## 1. 安装依赖（已完成）

```bash
cd frontend
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

## 3. 测试账号

- **志愿者**：volunteer / 123456
- **学院管理员**：admin / 123456

## 4. 功能说明

### 已实现功能

✅ **Excel模板下载**
- 平台管理员：儿童账号导入模板、学院管理员导入模板
- 学院管理员：志愿者导入模板、儿童账号导入模板

✅ **数据可视化图表**

- 已上架视频快捷监控（支持快捷下架）

✅ **平台Logo和主题**
- 红色主题风格（与Logo一致）
- Logo显示在所有页面头部
